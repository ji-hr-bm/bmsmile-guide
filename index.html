For New Members

신규 입사자님을 위한 안내

입사 초기에 알아야 할 필수 사항들입니다.

🖥 그룹웨어 초기 접속
🔐

초기 비밀번호

그룹웨어 ID는 회신주셨던 계정을 사용하시면 되고, 비밀번호는 111111 (1이 6개) 입니다. 최초 로그인 후 즉시 비밀번호를 변경해주세요.

📄 근로계약서 작성
✍️

카카오톡(모두싸인)으로 전송합니다

입사 시 제출하신 서류 및 협의된 처우를 바탕으로 카카오톡(모두싸인)을 통해 보내드립니다. '문서 확인하고 서명하기'를 클릭하여 전자서명을 완료해주세요.

📶 WiFi 이용 안내
📡

사내 WiFi

ID : bmsmile  /  PW : bmsmile!  — 회의실 비치 안내판에도 기재되어 있습니다.

📅 회의실 예약
🗓

그룹웨어(아마란스10) → 자원 → 자원예약

먼저 예약하신 분에게 사용권이 있으며, 서로 배려하며 예약 부탁드립니다.

🖨 사내 비품 / PC
📌

공용비품 위치

인사총무팀에서 수시로 구매하여 각 라운지 캐비닛 등에 비치해두고 있습니다.

🛒

개별 사무용품 구매

각 부서별 안내된 오피스디포 계정을 통해 구매 신청 가능합니다.

절차: 인사총무팀 슬랙 요청 → 확인 및 승인 → 오피스디포 구매 신청
🔧

PC 이상현상 발생 시

먼저 전원 및 연결(HDMI 등) 선을 확인해 주세요. 동일한 현상이 지속되면 인사총무팀에 문의해주세요.

🖨

복합기 토너 교체

복합기 최 하단에 토너 여분 보관됩니다.
① 하단 보관함 Open → ② 여분 토너 개봉 → ③ 토너 슬롯 Open → ④ 색상에 따라 교체 → ⑤ 교체된 토너 폐기

🚑 사내 구급함 위치
💊

라운지 뒷편 캔틴

진통제, 감기약, 지사제, 소화제, 밴드, 연고 등

💊

인사실 앞 캔틴

필요한 약이 있으시다면 인사실에 문의 주세요 :)

07
How to Apply

각종 신청 안내

모든 신청은 그룹웨어(아마란스10)를 통해 진행됩니다.

📅 연차 신청
🗓

그룹웨어 → 전자결재 → 휴가신청서

신규 입사자에게도 당해년도 발생연차를 미리 부여합니다. (예: 2025년 7월 1일 입사 → 연차 5개 생성)
연차/반차/반반차 제도 운영. 퇴직 시 미리 사용된 연차는 금액에 반영될 수 있습니다.

🙏 경조휴가 / 경조금 신청
📋

경조휴가 신청서

그룹웨어 → 전자결재 → 경조휴가신청서
제목: 경조휴가 신청의 건 / 비고: 경조휴가 종류

📋

경조금 신청서 (증빙서류 필수)

그룹웨어 → 전자결재 → 경조금신청서
결혼: 청첩장 / 출산: 출산확인증명서 / 사망: 가족관계증명서+사망진단서

🗓 휴일근로 대체휴일 신청
📋

그룹웨어 → 전자결재 → 휴일근무 신청서(휴일대체 동의서)

휴일근무일 기준 24시간 전 미리 상신 · 8시간 초과 불가
대체휴일 사용 시 '휴가신청서'도 별도 상신 필수 · 휴일근로 후 1개월 이내 사용

📜 증명서 발급
📄

재직/경력증명서

그룹웨어 → 인사/근태 → 증명서 → 증명서신청 → 등록

💳

급여명세서

그룹웨어 → 인사/근태 → 급여관리 → 급여명세서

📚 도서 구매 지원
📚

그룹웨어 → 전자결재 → 도서구입신청서

신청해주시면 확인 후 인사실에서 직접 주문해 드립니다.

🆔 사원증 분실 / 미지참
❗

사원증 미지참 출근 시

① 연락 가능한 팀원 또는 인사실 팀원에게 나와달라고 부탁해 우선 출입
② 출근 후 인사총무팀에 슬랙 반드시 전송 (출근 누락 방지)

🔁

사원증 분실 시

그룹웨어 → 전자결재 → '사원증 재발급 신청서' 상신 → 인사실 방문 후 수령

✏️

사원증 및 명함 수정/제작

커뮤니케이션파트에 슬랙 주시기 바랍니다.

💸 중소기업소득세 감면
💰

해당자 자동 적용 — 별도 신청 불필요

청년(만 15세~34세 이하), 60세 이상자, 장애인, 경력단절 여성의 경우 자동 적용. 인사실에서 연말정산 시 일괄 산정 반영합니다.
